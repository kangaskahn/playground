GIT NOTES

//COMMAND LIST

  • git init :: creates empty repository it folder
  • git status :: See how the repository status has changed
  • git add :: Adds FILENAME to the git tracker git add '*.txt' :: adds wildcards of all the same filetype git add * :: adds all the files
  • git commit -m '' :: Commits the changes from staging to the repository
  • git log :: "journal" of all the commits
  • git remote add :: Adds remote repository URL for pushes where = name of the remote
  • git push -u :: Pushes updates (The -u remembers the parameters so next time you run just run :: git push) git push -u origin master
  • git pull :: Pulls other people's commits and changes from the remote repository
  • git diff HEAD:: Tells what is different from our last commit (the HEAD shows the last commit you made)
  • git reset :: Unstage files
  • git branch :: Creates a new branch (without a branchname will show all branches [I think]) git branch -d :: deletes a branch (useful for doing clean ups)
  • git checkout :: Changes branches from one to the other
  • git rm :: Removes files from disk and will stage the removal of those files
  • git merge :: Merges one branch with the other. Make sure you are in the main branch (use git checkout )
